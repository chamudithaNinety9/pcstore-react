PC-Store E-commerce Website
===========================

Overview:
---------
PC-Store is a full-stack e-commerce web application designed for buying and selling a variety of computer products, 
including laptops, desktops, peripherals, and accessories. Built using Spring Boot for the backend and React for the frontend, 
PC-Store provides a seamless shopping experience with essential features for both customers and administrators.

Features:
---------
- Product Catalog: Browse and search products with details like name, brand, category, price, and images.
- Product Management: Admin dashboard for CRUD operations (create, read, update, delete) on products.
- Image Upload: Allows administrators to upload product images using a secure file upload mechanism.
- Search Products: Users can search for products by name, category, or brand for quick access.
- Responsive Design: Mobile-friendly layout ensuring accessibility across devices.
- Contact Us: Dedicated page for contacting support and viewing company details.

Technologies Used:
-------------------
- Frontend: React, Material-UI, React Router
- Backend: Spring Boot (Java), Spring Data JPA
- Database: MySQL
- File Uploads: Multer (Node.js middleware)
- Deployment: Heroku (Backend), Netlify (Frontend)
- Version Control: Git, GitHub
